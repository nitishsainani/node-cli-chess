Recommended Node Version - 14.*.*

npm install
npm start


- For changing the number of characters or board size, change in `index.js`.
- Initial Arrangement of Players goes like `column row`.
- CELL Notation is `<PLAYER>-<CHARACTER>-<CHARACTER NUMBER>`. (which is more efficient)
- Every move Notation is Like `<CHARACTER>-<CHARACTER NUMBER>:<MOVE>`
- Moves and characters are defined in constants file and new characters can be easily defined there
